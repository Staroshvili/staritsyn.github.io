Введение в фреймворк Vue

Что такое Vue.js?

Vue (произносится /vjuː/, примерно как view) — это прогрессивный фреймворк для создания пользовательских интерфейсов. В отличие от фреймворков-монолитов, Vue создан пригодным для постепенного внедрения. Его ядро в первую очередь решает задачи уровня представления (view), что упрощает интеграцию с другими библиотеками и существующими проектами. С другой стороны, Vue полностью подходит и для создания сложных одностраничных приложений (SPA, Single-Page Applications).

Установка фреймворка Vue

Фреймворк рекомендовано устанавливать через npm. Для начала обязательно убедитесь, что на вашем компьютере установлена версия Node.js не ниже 16. Установите или обновите, если необходимо. Далее вам нужно создать папку, открыть ее в терминале и выполнить следующую команду:

npm init vue@latest

В процессе установки вы должны указать название проекта, а также можете выбрать дополнительный инструментарий. Можете пока отказаться, выбрав No для инструментов, их всегда можно добавить позже.

После установки, для запуска фреймворка, вам нужно зайти в папку с вашим созданным проектом:

cd название проекта

Затем выполнить следующую команду, при дальнейших запусках выполнять ее не нужно:

npm install

Далее для запуска в режиме разработки выполните:

npm run dev

Появившийся в терминале линк, вы можете теперь открывать в браузере - по этому адресу и будет отображаться ваше VUE-приложение
